## Curso de Backend e API
Ministrado pelo Yoyo na plataforma Ingate
## Comandos para linux:
### buildar o container:
```bash
sudo docker-compose up --build
```
### Entrar no container (com ele rodando):
```bash
docker exec -it sr-api bash
```
### Executar os testes da API
```bash
npm run test
```
### Atualizar os módulos
```bash
npm run update
```
